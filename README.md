# ItGuru Portfolio

A modern, interactive developer portfolio website showcasing projects, experience, and skills.

## Project Structure

```
├── Background-locked/     # Background speaking/presentation section
├── Experience-page/       # Experience showcase with zoom effects
│   ├── periodic-exp/      # Periodic table-style experience display
│   └── ZoomExp-effect/    # Scroll-based zoom effects
├── Foreground-locked/     # Main portfolio components
├── Morph/                 # Morph animation effects
├── Projects-reel/         # Interactive projects carousel/reel
└── Deleted-elements/      # Archived/reference components
```

## Features

- Interactive scroll-based animations
- Projects reel with smooth transitions
- Zoom experience effects
- Modern responsive design
- Morph animations

## Technologies

- HTML5
- CSS3 (with modern animations and effects)
- Vanilla JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/navneet07/ItGuru-Portfolio.git
   ```

2. Open `Foreground-locked/devport.html` in your browser to view the portfolio.
